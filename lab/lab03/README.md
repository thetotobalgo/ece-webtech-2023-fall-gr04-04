
# ECE Web Technologies Corrections

A simple web server written in Node.js.

## Installation

Install application:

```
npm install 
```

## Usage

Start a web server with the command:

```
npm start
```

It will start a web server available in your browser at http://localhost:8080.

## Develop

Start a web server with the command:

```
npm run dev
```

It will start a web server available in your browser at http://localhost:8080.
